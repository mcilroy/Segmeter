# Segmeter
## Hotkey list
1. Main
- Open          Ctrl+O
- Open dir     Ctrl+Shift+O
- Save            Ctrl+S
- Next            Ctrl+Shift+Z
- Prev            Ctrl+Shift+A
2. Change the filters
- EXGR          Ctrl+Shift+F, 1
- NDI            Ctrl+Shift+F, 2
- None         Ctrl+Shift+F, 3

3. Tools
- Flood Fill   Ctrl+X
- Pincel        Ctrl+C

4. Color
- Yellow       Ctrl+Shift+1
- Green        Ctrl+Shift+2
- Whit          Ctrl+Shift+3
- Black         Ctrl+Shift+4
- Red           Ctrl+Shift+5
- Blue          Ctrl+Shift+6
