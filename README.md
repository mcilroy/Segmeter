# Segmeter
## Hotkey list
1. Main
- Open          Ctrl+O
- Open dir     Ctrl+Shift+O
- Save            Ctrl+S
- Next            Ctrl+Shift+Z
- Prev            Ctrl+Shift+A
2. Change the filters
- EXGR          Ctrl+Shift+F, 1
- NDI            Ctrl+Shift+F, 2
- None         Ctrl+Shift+F, 3

3. Tools
- Flood Fill   Ctrl+X
- Pincel        Ctrl+C

4. Color
- Yellow       1
- Green        2
- Whit         3
- Black        4
- Red          5
- Blue         6
